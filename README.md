# Le Choix Costaud

Première version mobile-first de la PWA.

## Fonctionnalités

- Accueil premium
- 6 catégories
- Catalogue de démonstration
- Recherche et filtres
- Favoris enregistrés sur le téléphone
- Comparateur jusqu'à 3 produits
- Questionnaire de recommandation
- Fiches détaillées
- Installation PWA
- Mode hors ligne simple
- Aucun lien affilié actif

## Mise en ligne

Déposez tous les fichiers à la racine d'un projet GitHub Pages, Netlify ou Vercel.

## Important

Les produits et prix de cette version sont des exemples génériques. Ils ne doivent pas être présentés comme des offres commerciales réelles.

Avant la monétisation :
- remplacer les exemples par des produits vérifiés ;
- ajouter les mentions légales et la politique de confidentialité ;
- déclarer clairement les liens affiliés ;
- vérifier les conditions de chaque programme partenaire ;
- régulariser le cadre administratif.
