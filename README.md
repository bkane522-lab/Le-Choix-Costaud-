# Le Choix Costaud — version publique finale

Contenu prêt à déployer :
- interface UX/UI validée ;
- 18 sélections dans 6 univers ;
- assistant de choix, recherche, filtres, favoris et comparateur ;
- recherches externes vers les marchands, sans affiliation active ;
- mentions légales, confidentialité et à propos ;
- PWA installable avec fonctionnement hors connexion ;
- icônes PWA aux formats réels 192 × 192 et 512 × 512.

## Déploiement
Remplacez l’ancien contenu du projet par tous les fichiers et dossiers présents ici, en conservant exactement leur arborescence, puis redéployez.
